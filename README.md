## Descrição

Segundo projeto em [Nest](https://github.com/nestjs/nest) baseado no curso da Alura.

## Dependências
- Node - Versão 12.20.0

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
